Installation

  1. Download a .icuewidget file using the Download button below
  2. In iCUE, navigate to the widgets section and click the + button
  3. Select the downloaded .icuewidget file
  4. The widget will appear in your widget picker

Visualizer Server Setup

The XE Visualizer and Pump Visualizer require a Python companion server that captures system audio and media info. A single server instance serves both widgets.

Requirements: Windows 10/11, Python 3.10+ (check "Add Python to PATH" during install)

  1. Download and extract NowPlayingServer.zip
  2. Double-click StartServer.bat to install dependencies and start the server
  3. The server will appear as an icon in your system tray. Right-click to quit.
  4. Set the Server Port in the widget settings to match (default: 16329)
