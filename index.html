Installation

  1. Download a widget ZIP using the Download button below (or grab the all-widgets bundle from Releases)
  2. Extract the contents into
    C:\Program Files\Corsair\Corsair iCUE5 Software\widgets\
  3. Restart iCUE
  4. Add the widget to your dashboard in iCUE

Visualizer Server Setup

The XE Visualizer and Pump Visualizer require a Python companion server that captures system audio and media info. A single server instance serves both widgets.

Requirements: Windows 10/11, Python 3.10+

1. Install dependencies:

pip install PyAudioWPatch numpy websockets winrt-runtime winrt-Windows.Media.Control winrt-Windows.Storage.Streams

2. Run the server:

python NowPlayingServer.pyw

3. Set the Server Port in the widget settings to match (default: 16329)
